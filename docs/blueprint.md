# **App Name**: SmartPlan

## Core Features:

- Dashboard: Display a clean, simple dashboard to view and organize daily activities.
- Calendar View: Provide a calendar view for a clear overview of upcoming events.
- Smart Scheduler: Use a tool to generate a schedule from natural language input.
- Smart Detail Detection: Use a tool to detect deadlines and task info from text input.
- Reminder Suggestions: Suggest optimal times to set reminders based on tasks.
- File storage: Store all task and reminder information in a file.
- Email Reminders: Send timely email reminders for upcoming events.

## Style Guidelines:

- Primary color: Soft blue (#89B6FF) to evoke trust and organization.
- Background color: Light blue (#F0F8FF), a very subtly tinted near-white, creating a clean, airy feel.
- Accent color: Light purple (#B09BFF) for interactive elements and highlights, creating a sense of calm and focus.
- Body and headline font: 'Inter', a sans-serif, for a modern, neutral look.
- Use clean, minimalist icons for tasks and categories.
- Responsive layout for use on various devices.
- Subtle transitions and animations for a smooth user experience.
